## Requirements
- Node 18
- npm 9.5.1 ->

## Setup
Copy the `.env.sample` to `.env` and fill it with the correct information.

## Running

I used npm.
```sh
# Install dependencies
npm install

# Run script with nodemon and restart on file change
npm run dev
```
yarn should also work.
```sh
# Install dependencies
yarn install

# Run script with nodemon and restart on file change
yarn dev
```
